Mess Management

Project Overview

Mess Management is a full-stack web application designed to streamline managing mess bills, user registrations, monthly reports, and admin operations. It features user-friendly interfaces for both members and administrators, making bill tracking and payments hassle-free.

Key Features

  • User registration and token-based authentication
  • Separate roles: Member and Admin
  • Monthly bill tracking and reports generation
  • Admin dashboard for managing users and bills
  • Responsive design with smooth navigation

Technologies Used

  • Python, Django & Django REST Framework
  • JavaScript (Vanilla), HTML5, CSS3
  • Responsive design and accessibility best practices
  • Google Fonts (Poppins)
