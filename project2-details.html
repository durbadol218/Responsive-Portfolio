FlowerWorld

Project Overview

FlowerWorld is an elegant and functional e-commerce platform for selling flowers online. It features user registration, flower browsing with category filters, a seamless cart and checkout experience, and an admin dashboard to manage inventory and orders.

Key Features

  • User registration and email activation
  • Browse flowers by category with filtering
  • 'Buy Now' button with order tracking
  • Admin dashboard for managing flowers and orders
  • Email confirmation after order and status update

Technologies Used

  • Python, Django, Django REST Framework
  • HTML5, CSS3, Vanilla JavaScript
  • Bootstrap 5 for layout
  • SMTP for email functionality
